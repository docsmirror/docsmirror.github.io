RDFFRS

Return predicate of succesfully loaded elements, setting the condition flags

This instruction reads the first-fault register (FFR) and places active elements in the corresponding elements of the destination predicate. Inactive elements in the destination predicate register are set to zero. This instruction sets the First (N), None (Z), and !Last (C) condition flags based on the predicate result, and sets the V flag to zero.

This instruction is illegal when executed in Streaming SVE mode, unless FEAT_SME_FA64 is implemented and enabled.

Encoding

RDFFRS <Pd>.B, <Pg>/Z

Decode for this encoding

if !IsFeatureImplemented(FEAT_SVE) then EndOfDecode(Decode_UNDEF); end; let g : integer = UInt(Pg); let d : integer = UInt(Pd); let setflags : boolean = TRUE;

Assembler Symbols

<Pd>

Is the name of the destination scalable predicate register, encoded in the "Pd" field.

<Pg>

Is the name of the governing scalable predicate register, encoded in the "Pg" field.

Operation

CheckNonStreamingSVEEnabled(); let VL : integer{} = CurrentVL(); let PL : integer{} = VL DIV 8; let mask : bits(PL) = P{}(g); let ffr : bits(PL) = FFR{}(); let result : bits(PL) = ffr AND mask; if setflags then PSTATE.[N,Z,C,V] = PredTest{PL}(mask, result, 8); end; P{PL}(d) = result;
